A simple **Todo List** app built with **React**, featuring animations, local storage persistence, and interactive UI elements.

## Demo

- Add, edit, delete, and mark todos as completed.
- Smooth animations on adding, updating, deleting todos using **Framer Motion**.
- Todos are persisted in **localStorage**, so they survive page refresh.

---

## Features

- Add new todos.
- Edit existing todos.
- Delete todos with confirmation.
- Mark todos as completed with a checkbox.
- Smooth animations when todos are added, updated, or removed.
- Todos are automatically saved in localStorage.

---

## Technologies Used

- **React** - Functional components and hooks (`useState`, `useEffect`)
- **Framer Motion** - Animations for adding/removing todos
- **React Hot Toast** - Notifications for actions (add, edit, delete)
- **Lucide React** - Icons for edit, delete
- **Tailwind CSS** - Styling
